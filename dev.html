Movie Download Website

Project Overview

This project is a static movie download website developed as part of our Web Technology subject. It allows users to download Bollywood and Hollywood movies and web series in multiple qualities such as 480p, 720p, and 1080p.

Features

  • Browse and download Bollywood and Hollywood movies for free.
  • Download web series in multiple qualities (480p, 720p, 1080p).
  • Clean, visually engaging design with a dark theme using HTML and CSS.

Project Structure

  • HTML Files: Structure and layout of the website.
  • CSS Files: Styles and visual aesthetics of the website.
  • Images Folder: Contains visuals like movie posters and thumbnails.
  • Links Folder: Houses download resources and links.
